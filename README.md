# Analisis-de-Datos---Segmentacion
Este proyecto realiza una exploración y análisis de datos del dataset global sobre tasas de suicidio, salud mental y trastornos por consumo de sustancias. Incluye limpieza automática, modelos de regresión y clasificación (Linear/Random Forest) y visualizaciones como correlaciones, importancia de variables y reducción de dimensiones (PCA).
